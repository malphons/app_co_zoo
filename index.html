Convex Optimization Zoo

Twelve windows into the geometry of optimal decisions

A function f is convex when the chord joining any two points on its graph never dips below the surface:

$$f\!\bigl(\theta\, x + (1{-}\theta)\, y\bigr) \;\leq\; \theta\, f(x) \;+\; (1{-}\theta)\, f(y), \qquad \theta \in [0,\,1].$$

This single inequality guarantees that every local minimum is global—there are no hidden valleys, no deceptive plateaus. The twelve exhibits below trace this idea from geometric roots to a real problem in robotics where every piece converges.

I. Foundations

A set $C \subseteq \mathbb{R}^n$ is convex when every line segment between two of its points stays inside. Supporting hyperplanes graze the boundary without crossing, certifying membership and separation.

The simplest convex objective is a quadratic bowl: least-squares fitting $\min_x \lVert Ax - b \rVert^2$ seeks its bottom via the normal equations $A^\top\! A\, x = A^\top b$. Projection asks the complementary question—find the nearest feasible point $\operatorname{proj}_C(x_0) = \arg\min_{x \in C} \lVert x - x_0 \rVert$—and regularization balances fidelity against simplicity through the penalty $\lambda \lVert x \rVert^2$, tracing the bias–variance trade-off as $\lambda$ varies.

II. Programs

A linear program $\min\{c^\top x : Ax \leq b\}$ optimizes over a polytope; solutions always live at vertices. A quadratic program $\min\!\bigl\{\tfrac{1}{2}\, x^\top Q\, x + c^\top x : Ax \leq b\bigr\}$ with $Q \succeq 0$ blends a curved objective with flat constraints. The Karush–Kuhn–Tucker conditions reveal which constraints are active—pinning the solution to the boundary—and which are slack.

III. Matrix Space

A symmetric matrix $P$ is positive definite ($P \succ 0$) when $x^\top\! P\, x > 0$ for every nonzero $x$; it defines the ellipsoid $\{x : x^\top\! P\, x \leq 1\}$. The set of all such matrices forms a convex cone—the matrix analogue of the positive real line.

Semidefinite programming constrains affine matrix functions to stay positive definite via linear matrix inequalities $F_0 + \sum_i x_i F_i \succeq 0$. Interior-point methods add the log-det barrier $\phi(X) = -\log\det X$, whose minimizer traces a central path to the constrained optimum. The LogDet divergence $D_{\mathrm{ld}}(P \| Q) = \operatorname{tr}(PQ^{-1}) - \log\det(PQ^{-1}) - n$ measures matrix mismatch in the intrinsic geometry of the cone—the entropic regularizer for the capstone.

IV. Synthesis

Every constrained convex problem has a dual: the primal seeks the best feasible point, the dual the tightest lower bound. The Lagrangian $\mathcal{L}(x,\lambda) = f_0(x) + \sum_i \lambda_i f_i(x)$ mediates between the two, and the KKT conditions—stationarity, feasibility, complementary slackness—certify optimality.

The capstone unites every thread: identifying a robot's inertial parameters from motion data. The decision variable is a positive-definite pseudo-inertia $\Phi$, the objective combines least-squares fit with LogDet regularization, and the constraints are LMIs enforcing physical consistency—one convex program synthesizing the entire zoo.
